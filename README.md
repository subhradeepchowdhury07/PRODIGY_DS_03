# PRODIGY_DS_03
# Decision Tree Classifier - Bank Marketing Dataset

## 📌 Overview
This project builds a **Decision Tree Classifier** to predict whether a customer will subscribe to a bank term deposit using the **Bank Marketing Dataset** from the UCI Machine Learning Repository.

## 🛠 Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## 📂 Files
- `bank-additional.csv`
- `DecisionTree_BankMarketing.py`
- `decision_tree.png`
- `confusion_matrix.png`

## 🚀 Features
- Data preprocessing
- Label Encoding
- Train-Test Split
- Decision Tree Classification
- Model Evaluation
- Decision Tree & Confusion Matrix Visualization

## ▶️ How to Run
1. Install dependencies:
```bash
pip install pandas matplotlib scikit-learn
```

2. Run the program:
```bash
python DecisionTree_BankMarketing.py
```

## 📊 Output
- Accuracy Score
- Classification Report
- Confusion Matrix
- Decision Tree Visualization

## 📖 Dataset
Bank Marketing Dataset - UCI Machine Learning Repository.
